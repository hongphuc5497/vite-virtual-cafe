## Interaction Contracts

### Timer Completion → Celebration Overlay

1. `useSessionTimer.timeLeft` reaches 0, `isRunning` set to `false` by the hook
2. `_index.tsx` detects `timeLeft === 0 && !isRunning` via `useEffect`
3. Session entry is built from current state: `appliedDurationMinutes`, detected vibe name, computed mood, `tracks` snapshot
4. Entry is written to `localStorage` key `virtual-cafe-sessions` (save-on-show per D-03)
5. `showCelebration` state set to `true`, `lastSession` state set to the entry
6. `CelebrationOverlay` renders with `fixed inset-0 z-50`, initially `opacity-0`, transitions to `opacity-100` (Tailwind `transition-opacity duration-500 ease-out`)
7. User taps anywhere on overlay backdrop → `setShowCelebration(false)`, overlay fades out
8. **Guard:** `showCelebration` flag prevents re-triggering on re-render. Only reset on explicit dismiss or new timer start.

### Audio Error → Inline Error + Retry

1. `HTMLAudioElement.play()` promise rejects OR `HTMLAudioElement.onerror` fires
2. `useAudioManager` catches the error, sets `trackErrors[label] = true`
3. `RoomMixControls` receives `trackErrors` from `audio.trackErrors`, passes `hasError` per track to `TrackControl`
4. `TrackControl` renders error badge icon (material "error", #ba1a1a, 14px, absolute top-right of track icon)
5. If error: retry button appears alongside pause button (material "refresh" icon)
6. User clicks retry → calls `onRetry` → `useAudioManager.retryTrack(label)` → calls `audioElement.load()` + `audioElement.play()`
7. On retry success: `setTrackErrors(prev => ({ ...prev, [label]: false }))` — error badge and retry button disappear
8. On pause toggle: error also clears (D-10)

### Relax Page PausedTracks Restoration

1. `relax.tsx` reads `SavedPreferences.pausedTracks` from localStorage (`STORAGE_KEY`)
2. Passes as `initialPausedTracks` to `useAudioManager(tracks, initialPausedTracks)`
3. Hook uses `initialPausedTracks` as the initial value for `pausedTracks` state instead of "all false"
4. **Guard:** A `useRef(loaded)` flag prevents the save `useEffect` from overwriting the restored state on initial mount render
